# Newark Medical Associates (Course Project)

Final term team project, the goal is to design a simple simulated medical management system.
Written in Java SpringMVC, a RESTful style api server.

### Enviornment
 - MySQL 5.6.37
 - Ubuntu 16.04
 - Java 1.8.0_151
 - Tomcat 8.5.24
 - Spring 4.2.0
